Installing Ginger-base

Ginger-base 2.1 is supported in Fedora 23, Ubuntu 15.10, Opensuse Leap 42.1 and Red Hat Enterprise Linux 7.2. As a WoK plug-in, WoK is needed to be installed for Ginger-base to run.

Fedora and RHEL:

First, download the files below if running F23:

WoK 2.2.0 Fedora 23 rpm
Ginger-base 2.2.0 Fedora 23 rpm

Or these files if running Red Hat Enterprise Linux 7.2:

WoK 2.2.0 RHEL 7.2 rpm
Ginger-base 2.2.0 RHEL 7.2 rpm

Then execute in a terminal:

sudo yum/dnf install <wok and ginger-base rpms>

Ubuntu:

First, download all the files below:

WoK 2.2.0 for Ubuntu 15.10 deb
Ginger-base 2.2.0 for Ubuntu 15.10 deb

Then execute in a terminal: 

sudo dpkg -i <wok and ginger-base debs>

openSUSE:

First, download all the files below:

WoK 2.2.0 for OpenSuse LEAP 42.1
Ginger-base 2.2.0 for OpenSuse LEAP 42.1

For Opensuse LEAP additional repos are needed to install dependencies:

sudo zypper ar -f http://download.opensuse.org/repositories/Virtualization/openSUSE_Leap_42.1 virt_repo
sudo zypper ar -f http://download.opensuse.org/repositories/home:/pxc:/winswitch/openSUSE_Leap_42.1/ winswitch_repo
sudo zypper refresh

If asked to trust the repositories keys, choose to 'trust always'. These are official Suse repos.

Finally:

sudo zypper install <wok and ginger-base rpms>

Retarting wokd service

After installing Ginger-base package, you just need to start WoK service to get it up:

sudo service wokd start

or:

sudo systemctl restart wokd.service

Note: The above command lines may vary depends on your operating system.

Then access WoK in your favorite browser https://<machine-ip>/8001

Have fun! =)
