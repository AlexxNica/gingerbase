Installing Ginger-base

Ginger-base depends on WoK. First step is to download and install it.

Afterwards, download a Ginger-base package according to your operating system.

We provide packages for the latest version of Fedora, openSUSE, Ubuntu and RHEL. Those packages were built on and for x86_64 machines.

After it, install the package using the default packaging management tool in your system. You can do it by command line like below:

Fedora and RHEL:

sudo yum install <ginger-base.rpm>

Ubuntu:

sudo dpkg -i <ginger-base.deb>

openSUSE:

sudo zypper install <ginger-base.rpm>

Retarting wokd service

After installing Ginger-base package, you just need to start WoK service to get it up:

sudo service wokd start

or:

sudo systemctl restart wokd.service

Note: The above command lines may vary depends on your operating system.

Then access WoK in your favorite browser https://<machine-ip>/8001

Have fun! =)
